## What This Is

A complete resource package for implementing an ISO/IEC 42001-aligned AI Management System (AIMS), covering all ten clauses of the standard's Plan-Do-Check-Act structure.

## How to Use

1. Complete the Context Worksheet first — it defines the AIMS scope everything else operates within.
2. Run the Gap Assessment Tool to baseline current conformity and prioritise build-out.
3. Approve the AIMS Policy Template alongside top management sign-off.
4. Populate the Risk and Opportunity Register and Objectives/KPI Template as ongoing living documents.
5. Run Internal Audits and Management Reviews on a planned cadence (the templates assume quarterly/annual, adjust to your organisation's audit programme).
6. Route every audit finding or nonconformity through the Corrective Action Procedure to closure.

## Relationship to the Rest of This Portfolio

ISO 42001 is a certifiable *management system* standard — it governs whether your overall AI governance process is sound, not any single system's compliance. It sits alongside, not beneath, the EU AI Act Compliance Toolkit (Project 7): a system can be ISO 42001-conformant at the process level while still needing its own EU AI Act conformity assessment. See `regulatory-standards-crosswalk.md` in the Reference Library for how the two relate.

The Core Governance Toolkit (Project 6) provides the operational Framework and Policy this AIMS scope references directly in the Context Worksheet.
